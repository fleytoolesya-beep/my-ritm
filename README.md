# Домашнее задание: схема БД музыкального сайта

Для сдачи задания используйте файл `music_site_schema.png`.

В схеме есть четыре таблицы:

- `genres` — музыкальные жанры;
- `artists` — исполнители;
- `albums` — альбомы;
- `tracks` — треки.

Связи между таблицами:

- один жанр может относиться ко многим исполнителям;
- один исполнитель может выпустить много альбомов;
- один альбом может содержать много треков.

Файл `generate_schema.py` нужен только для пересборки PNG, если схему
понадобится изменить.
