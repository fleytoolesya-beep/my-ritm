# Supabase Setup

## Что уже подготовлено

- файл переменных окружения: `.env`
- SQL-схема базы: `supabase/schema.sql`
- клиент Supabase для Python: `app/supabase_client.py`

## Что нужно сделать в панели Supabase

1. Открыть проект в Supabase.
2. Перейти в `SQL Editor`.
3. Создать новый запрос.
4. Вставить содержимое файла `supabase/schema.sql`.
5. Запустить запрос.

## Что это создаст

- таблицы для задач, целей, привычек, измерений и рабочих дней
- `profiles` для пользователей
- RLS-политики, чтобы каждый видел только свои данные
- триггер, который создает профиль после регистрации

## Следующий шаг

После применения схемы можно переходить к:

- регистрации и входу
- переносу текущих данных из локальной базы
- замене `SQLite` на облачное хранение
